# Python图书信息管理系统 python677

## 项目概述

Python图书信息管理系统是一个基于Django框架开发的系统，旨在帮助用户高效管理图书信息。项目包含了源码、开发文档、运行教程及开发环境配置。

## 技术栈

- 后端：Python（Django框架）
- 数据库：MySQL
- 前端：HTML、CSS、JavaScript

## 功能模块

- **用户认证**
  - 登录
  - 注册
- **书籍信息管理**
  - 新增/编辑/删除书籍
  - 书籍排序
  - 书籍搜索
- **出版社信息管理**
  - 新增/编辑/删除出版社信息
- **作者信息管理**
  - 新增/编辑/删除作者信息
- **分页查询**
  - 对书籍、出版社、作者信息进行分页显示

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 安装Python及Django框架
2. 配置MySQL数据库
3. 克隆源码至本地
4. 根据开发文档配置环境
5. 执行数据库迁移命令
6. 运行开发服务器

## 目录结构

```
├── apps                # Django应用目录
```
